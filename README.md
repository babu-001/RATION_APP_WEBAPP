# RATION_APP_WEBAPP

##  Project Overview
This is a Flask-based web application that predicts ration shop demand using machine learning.

##  Features
- Predicts demand using trained ML model
- Uses real government dataset
- Helps in resource planning and decision making

##  Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy

##  How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://localhost:5000

##  Model Info
- Algorithm: Random Forest
- Dataset: TN PDS Data
